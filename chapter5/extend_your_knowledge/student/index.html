Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

The sticky position allows an element to remain fixed at the top of the page as you scroll past it.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

This box is relatively positioned, meaning it's shifted 90px down and 30px to the right from where it would normally appear in the flow of the document.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

This box is positioned relative to the browser window and stays fixed in place even when scrolling. It appears 90 pixels from the top and 10 pixels from the right.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

This box is positioned absolutely within its nearest positioned ancestor. It appears at the bottom-right corner with 0 pixels from both the bottom and right edges.
